### 3.1语法

--------------

1. 区分大小写
2. 标识符
    - 标识符是指变量，函数，属性的名字，或者函数的参数，有以下格式
    ```text
    第一个字符必须是一个字母，下划线(_)或者一个美元符号($)
    其他字符可以是字母，下划线，美元符号或数字
    ```
    - 按照小驼峰命名法，如myCar
3. 注释
    ```javascript
    //单行注释
    
    /*
    * 多行注释
    * */
    ```
4. 严格模式
    - 在脚本中启动严格模式在顶部添加如下代码
    ```javascript
    "use strict";
    ```
    - 它是一个编译指示
    - 也可以指定函数在严格模式下执行
    ```javascript
    function doSomething () {
       "use strict";
       //函数体
    }
    ```
5. 语句
    ```text
    一个语句以分号结尾，可以不要分号但是不推荐这种做法
    推荐始终使用代码块即使代码块中只有一条语句
    ```
